## 抽奖流程：

1. 选择当次要抽奖的个数（22、12、6、2），默认选择2。
2. 点击『开始』按钮，进入抽奖状态（这个过程仍可修改抽奖个数）。
3. 点击『抽奖！』按钮，生成抽奖结果。
4. 点击任意个数按钮，可以回到闲置状态，已中奖的用户标记为红色，不会二次命中。

## PS：

* [ ] PC网页端：

* 滚动鼠标滚轮，可以放大或缩小球体；
* 鼠标单击有爱心特效；
* 鼠标可与蜘蛛网特效交互。

* [ ] 手机移动端：

* 按钮有点击爱心特效；
* 触摸可与3D球特效有交互。
